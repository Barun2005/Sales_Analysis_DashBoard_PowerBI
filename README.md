# 📊 Super Sales Store Dashboard

## Overview
An interactive Power BI dashboard built on 10,000+ US superstore 
retail records, designed to surface sales trends, profit drivers, 
and regional performance for data-driven decision-making.

## Pages
- **Page 1 — Sales Overview:** KPIs, regional filters, YoY trends, 
  payment/segment/category breakdowns, geographic map
- **Page 2 — Forecast:** 15-day sales forecast with confidence 
  interval using Power BI's built-in analytics

## Key Metrics Tracked
- Total Sales: 450K+ | Profit: 53.4K | Quantity: 6,251
- Average Delivery Days: 3.91
- YoY Sales & Profit Growth (2019 vs 2020)

## Tech Stack
| Tool | Purpose |
|------|---------|
| Power BI Desktop | Dashboard & visualizations |
| Power Query | Data cleaning & transformation |
| DAX | KPI measures & calculations |
| Excel | Source data preparation |

US Superstore Sales dataset — 10K+ records across
